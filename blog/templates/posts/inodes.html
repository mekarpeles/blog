'No space left of device' but disk isn't full?

Mek, September 19, 2012

Possibility 1: Out of inodes

One possible reason for encountering a 'No space left on device' error is your machine has run out of available inodes.

What is an inode?

Many Unix file systems use data structures called inodes whose purpose is to store file metadata. Each file refers to its inode data via an i-number (inode number). The inode data structure typically stores information like about files and directories (folders), such as file ownership, access mode (read, write, execute permissions), and file type.

On many types of file system implementations, the maximum number of inodes is fixed at file system creation, limiting the maximum number of files the file system can hold. A typical allocation heuristic for inodes in a file system is one percent of total size. [1]
